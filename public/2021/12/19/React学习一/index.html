一、React是什么

React元素不可修改，更新的方法只能是创建一个新的元素传入ReactDOM.render

React 只更新它需要更新的部分,尽管每一秒我们都会新建一个描述整个 UI 树的元素，React DOM 只会更新实际改变了的内容，也就是例子中的文本节点。
